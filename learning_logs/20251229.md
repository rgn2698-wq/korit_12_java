2. conversion(형변환): 특정 변수를 다른 자료형으로 변환시키는것.
   upcasting: 특정 변수를 더 큰 용량의 자료형으로 변환시키는것.
    임시적으로 이루어짐: 작은 용량에서 큰 용량으로 변환시킬때는 데이터의 값을 변질되지 않기
    때문에 직접적으로 자료형을 명시하지않아도됨

```java
public static void main(String{} aegs) {
    // 변수 선언 및 초기화
    char cast1 = 'A';
    // 1. 업캐스팅 #1
    System.out.println("변형 : " + (int)cast1; // 결과값 : 65
    // 2. 업캐스팅 #2
    int cast2 = 97;
    // 3. 다운캐스팅 #1
    System.out.println("변형 : " + (int)cast2; // 결과값 :a
    // 3. 다운캐스팅 #2
    char casted2 = (char)cast2; // 이 부분이 '명시적' - 업캐스팅 #2와의 차이점 확인
   
}
```
4. Operaton(연산자)
    1. 대리 연산자 "+" : "+" 왼쪽에 있는 변수에 '+' 오른쪽에 있는 데이터(값)을 '대입'한다는
    의미 - 즉 a=b라고 했을때 a와 b의 값이 같다 라는 의미가 아니라는 점에 주목해야합니다.
        -같다고 이야기 하고싶다면 '=='를 씁니다. 근데 Javascript에서는 '=='도 있고 '==='도
         있습니다.
    2. ' ' : 수학이랑 똑같습니다. -문자, 문자열 자료형의 경우에도 + 연산자는 적용이 가능.
    3. % : 나머지 연산자 - 나누고 나서 남는 나머지를명시함. 즉 결과값은 int. long이여야 함.
    4. 복합대입연산자
        1.+=
        2.-=
        3.*=
        4./=
    5.'++ / --'

```java
public static void main(String{} aegs) {
    int num = 1;
    System.out.println(num++);
    System.out.println(num);
    System.out.println(++num);
    System.out.println(num==);
    System.out.println(num==);
    System.out.println(num--);
    System.out.println(num);
    System.out.println(--num);
}
```
